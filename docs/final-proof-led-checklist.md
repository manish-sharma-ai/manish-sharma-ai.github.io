# Proof-led release checklist

- [ ] Public case source is verified, production-safe, and referenced through `externalLinks.ts`.
- [ ] Imported image exists in the registry with source path, alt text, caption, and Exafuse credit.
- [ ] Every metric has source context and a limitation.
- [ ] No company execution or personal contribution is inferred.
- [ ] Monitoring is not presented as final quality or release approval.
- [ ] Homepage order is proof → Cockpit → method.
- [ ] Reference menu stays compact and deeper routes remain discoverable.
- [ ] Imported assets stay within budget and include responsive derivatives.
- [ ] `npm run check`, `npm run build`, `npm run audit:all`, and `git diff --check` pass.
- [ ] The external source repository remains untouched.
