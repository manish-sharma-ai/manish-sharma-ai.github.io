# Page choreography target map

The target map groups existing semantic sections into a small number of visual chapters. It does not remove headings, landmarks, anchors, evidence boundaries, or route content.

## Final verification

- Baseline estimated visible chapters: **346** across 58 routes.
- Final visible chapters: **233** across 58 routes.
- Homepage, Public Work, and Tools each resolve through five deliberate chapters.
- `/404` resolves through two chapters.
- Every other visitor-facing route resolves through four family-level chapters, even where more semantic sections remain in the document.
- The exact per-route baseline count, final count, scoring, implementation, and remaining constraint are recorded in `page-choreography-matrix.csv`.
