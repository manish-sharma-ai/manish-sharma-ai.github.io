# Public proof-story standard

Each `/public-work/exafuse/[slug]/` page is a Manish Sharma Lab interpretation of a public Exafuse case context. It must include:

1. an Exafuse ownership boundary;
2. real attributed lead media;
3. industrial problem and public process route;
4. only source-bound public facts;
5. evidence and validation context;
6. a specific “what it proves” list;
7. a specific “what it does not prove” list;
8. a clearly labeled Manish Sharma Lab interpretation;
9. a related decision artifact; and
10. the exact public Exafuse source plus the Exafuse commercial route.

The case page does not duplicate an Exafuse article. It uses concise paraphrase, structured facts, and an explicit limitation to help a reader ask a better next question.

Author schema describes Manish Sharma only as author of the interpretation. Exafuse remains the organization and public case source.
